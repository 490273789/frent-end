# 线性结构-栈

> 可基于数组/链表实现，栈可以看作是一种受限制的数组/链表。

## 特点

1. 受限的线性结构
2. 后进先出 - Last In First Out

## 操作

- 入栈：push O(1)
- 出栈：pop O(1)
- 访问栈顶：peek O(1)

## 属性

1. 栈底：栈的最下面的元素
2. 栈顶：栈的最上面的元素
